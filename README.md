# 🌍 StayZiv Project

A full-stack travel listing web application where users can explore, create, and review travel stays.

---

## 🚀 Features

* 🔐 User Authentication (Signup/Login/Logout)
* 🏡 Create, Edit, Delete Listings
* ⭐ Add Reviews on Listings
* 🔒 Authorization (Only owner can edit/delete)
* 💬 Flash messages for user feedback
* 📱 Responsive UI using Bootstrap

---

## 🛠️ Tech Stack

* Backend: Node.js, Express.js
* Database: MongoDB, Mongoose
* Frontend: EJS, Bootstrap
* Authentication: Passport.js
* Other: Method-Override, Connect-Flash, Express-Session

---

## 📂 Project Structure

StayZiv-project/

* models/        → Mongoose schemas
* routes/        → All routes
* views/         → EJS templates
* public/        → Static files
* utils/         → Utility functions
* middleware.js  → Custom middleware
* app.js         → Main server file

---

## ⚙️ Installation & Setup

1. Clone the repository
   git clone https://github.com/vishwas-arora18/StayZiv-project.git

2. Go to project folder
   cd StayZiv-project

3. Install dependencies
   npm install

4. Start the server
   node app.js

5. Open in browser
   http://localhost:8080
